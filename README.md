## 安装教程

初学者不需要安装，安装是参与共建或者做二次开发才需要的步骤。

首先fork主仓库到文件夹，然后执行下面的命令。
1.  npm install -g static-server
2.  static-server
3.  http://localhost:9080/frontpage/

## 如何共建
